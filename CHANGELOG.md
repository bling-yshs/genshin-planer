# Genshin Planner 更新日志

## [Unreleased]

### Fixed

- 修复多个角色养成计划中材料充足性计算不正确的问题，现在材料消耗会按照角色顺序累计计算
- 移除二维码登录弹窗和删除确认弹窗的毛玻璃效果，改用纯色背景提高可读性

### Changed

- 自动生成的养成计划名称中，超过3个角色时使用"等"代替"..."

## [0.1.0] - 2026-01-18

### Changed

- **⚠ 破坏性更新**
  - 此次更新没有验证之前数据的兼容性

- **迁移数据源从 玉衡杯 到 hakush**
  - 旧数据源已停止更新，全面切换到 hakush 数据源
  - 角色列表、材料数据、物品信息均从新数据源获取

- **快速添加角色材料后，直接在养成计划中展示角色卡片，而不是展示材料列表**

- 重新设计 UI 主题为原神风格
    - 新增玻璃态效果（glassmorphism）和渐变背景
    - 新增元素发光效果、渐变文字、悬浮动画、星光闪烁等特效
    - 优化所有卡片、按钮、输入框、对话框的视觉效果

- **养成计划自动创建优化**
    - 通过快速添加角色材料时，自动创建正式计划
    - 计划名称为角色名（用下划线连接，最多3个，超出用...表示）
    - 通过左侧物品库添加时，自动创建"未命名的养成计划"
    - 创建的计划可以正常改名、删除、保存
    - 不再出现"未保存的计划"临时状态

- **角色选择器 UI 优化**
  - 已选择的角色列表从顶部移至底部，
  - 添加双层动画效果：容器展开/收起 + 角色卡片弹出/缩小

- **性能优化**
  - 左侧物品库改为虚拟列表，增强页面性能

## [0.0.5] - 2026-01-03

### Added

- 初始版本发布
- 支持米哈游扫码登录
- 支持角色/武器材料计算
- 支持养成计划管理
